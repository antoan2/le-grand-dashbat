# Le Grand Dashbat

## Installation
Création d'un env virtuel, et installation des requirements :

  mkvirtualenv -a . --python=3.8 grand-dashbat
  pip install -r requirements.txt
  pre-commit install
  
On peut maintenant lancer l'application :

  python app/app.py
